dnkscr_f(userInput, depth, proctype, swa, req, res, rootkeys, dispout) → {object}

Direct or derivative noun root keys inference score count.
Parameters:
Name Type Description
userInput string Input sentence.
depth number Derivative noun search level depth parameter arguments (1-3).
proctype number Specifies the type of processing to be done, either 0 or 1. A value of 0 means that repeated keys will not be counted, while a value of 1 allows repeated keys to be counted.
swa number An integer in the range of 0-2, which determines the amount of stop words to be processed. 0 processes the minimum amount, 1 adds extra stop words, and 2 includes the maximum number of stop words from three distinct lists.
req array Requests.
res array Responces.
rootkeys json Rootkeys.
dispout boolean (true/false) Display processing output results in terminal.
